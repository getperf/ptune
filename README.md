# ptune  

Google Tasks と連携して作業時間を記録するポモドーロタイマーアプリ

**ptune** は、日々の作業を「計画 → 集中 → 振り返り」まで  
一連の流れとしてサポートする個人向けタイマーアプリです。

スマホだけでタスク管理と作業ログを完結でき、  
Obsidian と合わせて使うことで 1 日の記録を自然に蓄積できます。

## 主な機能と利用イメージ（1 日の流れ）

### 1. タスク同期  
- Google Tasks と双方向で同期  
- 今日のタスクがスマホに自動で反映  
- 完了状態も Google 側へ即時反映

**利用イメージ：朝 — 今日やることを確認**  
- Obsidian / ptune-log で整理したタスクがスマホに届く  
- すぐ作業を始められる

---

### 2. ポモドーロタイマー  
- 25 分作業＋休憩のサイクル  
- タイマーの開始・停止がワンタップ  
- 集中モードで作業に没頭しやすいデザイン

**利用イメージ：昼 — 集中して作業を進める**  
- タスクを選んでポモドーロを開始  
- 作業ログが自動で紐づく  
- 完了した作業は Google Tasks に反映される

---

### 3. 作業ログの記録  
- ポモドーロの実行履歴を自動保存  
- 作業時間の合計・履歴を一覧で確認  
- 記録した内容は後の振り返りや分析に利用可能

**利用イメージ：夜 — 1 日の振り返りに活用**  
- どのタスクにどれだけ時間を使ったかがひと目で分かる  
- Obsidian の ptune-log が作業ログを取り込み、  
  1 日のまとめレポートを自動生成

## 連携構成（全体像）

```
Obsidian / ptune-log
（朝：タスク作成 → 夜：振り返り）
        │  朝：タスクをエクスポート
        ▼
Google Tasks
        ▲
        │  夜：作業ログをインポート
        ▼
ptune（スマホ）
（昼：作業を実行・記録）
```

3 つのツールを組み合わせることで、  
日々のタスク管理と学習ログが自然に蓄積されます。
