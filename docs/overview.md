# ptune overview

## Purpose

ptune is a Flutter application for pomodoro-based task execution and review.

Core themes:

- task management
- pomodoro timing
- Google Tasks integration
- local cache for continued use during connectivity problems

## Current maintenance stance

The project is maintained mainly through:

- bug fixes
- small refactorings
- operational adjustments

Large redesigns are not the default direction.

## Main functional areas

- authentication
- task list and task editing
- local and remote task synchronization
- pomodoro timer and review flags
- export and sync support features
